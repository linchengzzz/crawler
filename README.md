# HQQ App下载抓取

> Env：Node.js Git 

## 项目下载安装

```shell
git clone https://github.com/linchengzzz/crawler.git
cd ./crawler
npm install
```

## 项目运行

- window: `npm install` 完成之后双击 `run.bat` 即可运行
- MacOS/Linux: `npm install` 完成之后 运行指令 `npm start` 或 `node ./app.js`
